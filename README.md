# gor-om-mig
Make a web page with dated styling/layout more modern and UX/UI friendly

## Original site
(https://gatesnfences.com/)

## Om sidan
Sidan jag utgick ifrån har otroligt mycket innehåll. Det var omöjligt att få med allt, utan jag fick så klart prioritera att
"viktigaste" fanns med. Jag hade gärna haft med t ex leverantörer/märken kanske i ännu en kolumn på sidan vid helskärm och under artiklarna på
mindre skärmar. Även en bättre kontaktsida med formulär. Men det hanns helt enkelt inte med. 

Jag valde att behålla deras gamla header och plockade upp lite orangea detaljer från loggan, som färg på länkar i artiklar och som hovereffekt
i menyerna. Detta för att göra det lite mer enhetligt och "brandat" men utan att det känns rörigt.

Jag tänkte att på startsidan vill man visa vad man sysslar med och locka kunden med nyckelord som "kvalité", "customized", att det är ett hantverk
och så vidare. Jag tog saker som stod på originalhemsidan och försöke krama ur essencen av budskapen och vad de har att erbjuda.

About-sidan blev mycket text, och rakt kopierad från originalet. Men det blir mer av ett aktivt val för användaren att ta del av så mycket text.
Jag hade gärna plockat ut mer väsentlig information om företaget istället för att bara skriva om aluminium, men det hann jag inte heller.

På de två produktsidorna tog jag texten som fanns på originalsidan, men la en stor del av texten som dropdown under "everything about our....", så
att användaren snabbare kan se själva produkterna istället för att läsa om dem. Produktkorten hade lämpat sig bättre med grid istället för flexbox, men det upptäckte
jag lite för sent. Produktkorten är länkar så man kan trycka var som helst på kortet för att komma till produkten.

Överlag har jag försökt skala ner och visa det viktigaste först, inte överösa användaren med all information på en gång, samt jobbat mycket på det responsiva.
Jag har även försökt göra koden så smidig som möjligt, men det var inte helt enkelt att försöka hitta variabler eller mixins som går att applicera rakt av på
olika element och faktiskt förenkla. Men det är väldigt smidigt när det fungerar!
